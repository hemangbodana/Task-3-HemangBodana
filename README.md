# Task-3-HemangBodana
Task-3 
Aim - Digital Matchmaker that recommends items based on user preferences using content-based filtering

The project utilizes an Input-Process-Output (IPO) model:

Input: Capture user states, such as explicit profile tags or implicit click history.

Process: Apply mathematical similarity logic to compare user profiles against item attributes.

Output: Generate a ranked "Top-N" list of relevant matches to solve choice overload.
